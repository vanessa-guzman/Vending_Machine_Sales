# Vending Machine Sales

**Tools:** SQL, Tableau, Google Slides <br />
**SQL Functions and Statements:** STR_TO_DATE, ALTER, UPDATE, LIKE, HAVING, Subqueries, Aggregation Functions <br />
**Skills:** data cleaning, data wrangling, data analysis, data visualization (in Tableau) <br />
**Objective:** Understand trends in customer behaviors and trends based on location. <br />
<br />
**About the data:** Transaction data (e.g. product, transaction date, price) was collected from vending machines at four locations in New Jersey. Data was collected between January and August of 2022. The data can be found on [kaggle](https://www.kaggle.com/datasets/awesomeasingh/vending-machine-sales). <br />
<br />

**What was done:**
* Converted dates from string format to DATETIME format
* Filled in missing data
* Provided descriptive statistics of products sold
* Analyzed monthly sales 
* Analyzed sales by weekday
